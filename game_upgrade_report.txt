HILL CLIMB GAME UPGRADE REPORT
==============================

1. VISUAL UPGRADES
------------------
- **Car Rendering**: Now drawn with separate chassis, cabin, and rotating wheels.
- **Suspension**: Added vertical bounce (sine wave based on velocity) for "floaty" feel.
- **Terrain**: Implemented multi-layered sine wave generation for "rolling hills" effect.
- **Parallax**: Added background hill layer moving at 30% speed for depth.
- **Environment**: 
  - Dynamic Sky Gradient (Day -> Night cycle based on distance).
  - Procedural Stars appear at night.

2. PHYSICS & FEEL
-----------------
- **Gravity**: Set to 0.1 (Gentle).
- **Acceleration**: 0.05 / Friction: 0.02 (Smooth ramp).
- **Rotation**: 
  - Calculated based on 40px forward lookahead.
  - Damped to prevent jitter.
  - Clamped to +/- 0.6 radians to prevent flipping.
- **Mobile Controls**:
  - `touchstart` + `mousedown` triggers acceleration.
  - `touchend` + `mouseup` triggers coasting.
  - Spacebar support added for desktop.
